# 📡 Network Traffic Analysis using Wireshark

## Objective
To analyze network traffic and identify suspicious activity using Wireshark.

## Tools Used
- Wireshark

## Steps Performed
- Captured live network packets
- Filtered traffic using protocols (HTTP, DNS, TCP)
- Analyzed packet details such as source/destination IP
- Identified unusual traffic patterns

## Key Learnings
- Packet structure and protocols
- Network behavior analysis
- Basics of intrusion detection

## Outcome
Successfully analyzed 1000+ packets and identified potential suspicious traffic.
