# 🔍 Port Scanning & Vulnerability Detection using Nmap

## Objective
To identify open ports and potential vulnerabilities in a system.

## Tools Used
- Nmap

## Steps Performed
- Performed TCP SYN scans
- Identified open ports and services
- Used service version detection
- Analyzed potential security risks

## Key Learnings
- Understanding open ports and services
- Basics of system vulnerabilities
- Attack surface identification

## Outcome
Scanned 50+ ports and identified multiple open services on local systems.
